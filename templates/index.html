CLI Usage
Available Commands:

python cli.py scan example.com --tools amass subfinder httpx --analyze

# List all scans
python cli.py list

# View scan details
python cli.py show <scan_id>

# Run AI analysis on existing scan
python cli.py analyze <scan_id>

# Generate graph visualization
python cli.py graph <scan_id>

# Delete a scan
python cli.py delete <scan_id>
